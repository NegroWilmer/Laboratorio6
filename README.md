# Laboratorio6

Ejercicio # 1
Una institución educativa desea modelar su sistema de personas vinculadas a la
organización. Todas las personas comparten características generales, pero tienen roles
distintos.
Se requiere modelar lo siguiente. Los Estudiante tienen como atributos identificación,
nombre, edad, código, y un método para mostrar información. Un Docente tiene como
atributos identificación, nombre, edad, especialidad y salario y un método mostrar
información. Un Administrativo tienen como atributos identificación, nombre, edad, cargo y
dependencia y un método mostrar información.
Además, existe un método calcularPago(),para docentes y administrativos y retorna el salario
a pagar, el salario de un docente se calcula así: numero de horas cátedra por el valor de la
hora cátedra menos el 12% de deducciones, el salario de un administrativo se calcula así:
salario fijo mensual menos 12% de deducciones.
Se Requiriere:
1. Diseñe un diagrama de clases UML.
2. Implementación Lenguaje de programación java

Ejercicio # 2.
Una empresa de transporte desea desarrollar un sistema para gestionar diferentes tipos de
vehículos. Todos los vehículos comparten ciertas características, pero también tienen
comportamientos específicos según su tipo.
Un Carro tiene los siguientes atributos placa, marca, velocidad máxima y numero de puertas,
una Moto tiene los siguientes atributos placa, marca, velocidad máxima y tipo ( deportiva,
scooter,etc), un Camión tiene los siguientes atributos placa, marca, velocidad máxima y
capacidad de carga.
Todos los vehículos deben implementar un método llamado mover() que describe cómo se
desplazan, pero cada tipo lo implementa de manera diferente.
Además, se requiere un método calcularPeaje() que también debe comportarse de forma
distinta según el tipo de vehículo.
Se Requiriere:
1. Diseñe un diagrama de clases UML.
2. Implementación Lenguaje de programación java.
